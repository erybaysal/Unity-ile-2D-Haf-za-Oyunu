# Unity ile 2D Hafiza Oyunu
### Oyunun amacı 4 işlemi ve hafızanı kullanarak hedef sayıya ulaşmak. Kullanıcı kapalı olan kartlardan 2 adet seçer ve bu kartlara uygulamak istediği işlemi seçer. Eğer yaptığı seçimler doğrultusunda hedef sayıya ulaşırsa kartlar açık kalır ve hedef sayı değişir.
### Ulaşamazsa ise kartlar 1 saniyeliğine kırmızıya döner ve kapanır.
